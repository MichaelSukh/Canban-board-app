# Инструкция по развертыванию проекта

В данном документе описаны шаги для развертывания проекта Canban-board-app как на удаленном сервере, так и на локальной машине.

## Развертывание на сервере

Вам необходимо иметь арендованный VPS/VDS и зарегистрировать домен на IP сервера. Рекомендуемые сервисы:

**Для аренды VPS:**
* [AdminVPS](https://my.adminvps.ru)
* [Hip Hosting](https://hip.hosting/ru)

Для корректного развертывания на сервер необходимо установить **Ubuntu 24.04**.

**Для регистрации домена:**
* [SpaceWeb (Sweb)](https://sweb.ru/)

После успешного запуска сервера и регистрации домена подключитесь к серверу по SSH и выполните следующие команды:

```bash
git clone https://github.com/MichaelSukh/Canban-board-app.git
cd Canban-board-app
chmod +x ./deploy.sh && ./deploy.sh
```

После того как скрипт установит Docker и Docker Compose (если они еще не установлены на сервере), необходимо будет ввести данные для конфигурации:
* Ваш домен
* Email для получения сертификатов от Let's Encrypt
* Имя пользователя базы данных
* Пароль для базы данных
* Название базы данных

Затем просто дождитесь завершения сборки и запуска Docker-контейнеров.

## Развертывание на локальной машине

Для развертывания проекта локально выполните следующие шаги.

Сначала клонируйте репозиторий:

```bash
git clone https://github.com/MichaelSukh/Canban-board-app.git
```

### Запуск backend

1. Перейдите в директорию бекенда:
   ```bash
   cd Canban-board-app/backend
   ```
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   ```
3. Активируйте виртуальное окружение:
   * **Linux/macOS**:
     ```bash
     source venv/bin/activate
     ```
   * **Windows**:
     ```powershell
     .\venv\Scripts\activate
     ```
4. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
5. Создайте конфигурационный файл `.env` в директории `backend` и заполните его. В качестве примера используйте файл `example.env`, который уже есть в этой директории.
6. Убедитесь, что сервер PostgreSQL запущен. После этого примените миграции базы данных:
   ```bash
   alembic upgrade head
   ```
7. Запустите сервер:
   ```bash
   python run.py
   ```

### Запуск frontend

1. Откройте новый терминал и перейдите в директорию фронтенда (или вернитесь в корень и затем в `frontend`):
   ```bash
   cd Canban-board-app/frontend
   ```
2. Установите необходимые пакеты:
   ```bash
   npm install
   ```
3. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
